# CRUD Inventario - Python (Consola)

Sistema de gestión de inventario en consola.

## Funcionalidades
- Alta de productos
- Modificación
- Eliminación
- Búsqueda
- Categorías
- Persistencia de datos (si tiene)

## Tecnologías
- Python 3.x

## Objetivo
Proyecto práctico para afianzar lógica, estructuras de datos y operaciones CRUD.
